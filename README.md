# qw

## Locs
copy locs to `qw/locs/`

## Config
**Save/setup**

1. load your current config.
2. copy `tvs_alias.cfg` and `tvs_tp.cfg` to `id1/`
3. edit player settings and binds in `tvs_tp.cfg`.
4. start ezquake
5. `/exec tvs_tp.cfg`
6. `cfg_save xx`

Config is now saved at `ezquake/configs/xx.cfg`.

**Load config**

```
/cfg_load xx
```
