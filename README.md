# qw

## Locs
copy locs to `qw/locs/`

## Config
**Save/setup**

1. load your current config
2. copy `tvs_alias.cfg` and `tvs_tp.cfg` to `id1/`
3. edit binds in `tvs_tp.cfg`
4. `/exec tvs_tp.cfg`
5. 
```
cfg_save_aliases 1
cfg_save_binds 1
cfg_save_userinfo 2
cfg_save xx
```

**Load config**

```
/cfg_load xx
```
